PROJECT TITLE: Creature War

PURPOSE OF PROJECT: To create a battle between a good and bad armies. The armies are made up of subclasses of the Creature class.

VERSION or DATE: 11-10-2020

HOW TO START THIS PROJECT: Create an instance of a War class and then call its battle method.

AUTHORS: Greg Babbert and Bill Crosbie

USER INSTRUCTIONS: Create an instance of a War class and then call its battle method. This will tell you which side won the war.
